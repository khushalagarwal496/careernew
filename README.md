# Career Compass 🧭

Career Compass is an AI-powered platform designed to help students navigate their career paths. It provides personalized alerts for jobs, internships, hackathons, and learning resources, bridging the gap between talent and opportunity.

## 🚀 Features

- **Job & Internship Alerts:** Real-time updates on relevant openings.
- **Hackathon Finder:** Discover coding competitions and events.
- **Learning Resources:** Curated roadmaps for skill development.
- **User-Friendly Interface:** Built with a modern and responsive design.

## 🛠️ Tech Stack

This project is built using modern web technologies:
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Deployment:** Vercel
